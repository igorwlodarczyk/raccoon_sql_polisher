# 🦝 Raccoon SQL Polisher
A formatter for PostgreSQL SQL queries that enhances readability and enforces a consistent coding style.
## 🦝 Setup
``` bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 🦝 Usage
``` bash
sqlraccoon <PATH>
```

## 🦝 Tests
``` bash
pytest tests/
```
